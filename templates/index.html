Travel Itinerary API

Overview

This API generates a structured travel itinerary for a given destination using a large language model. The response is returned as JSON with a fixed schema.

API Endpoint

GET http://localhost:8000/api/v1/itinerary

Query Parameters

  • destination (required): The destination to generate an itinerary for.
    Example: destination=Kyoto

Example Request

curl "http://localhost:3000/api/v1/itinerary?destination=Lisbon"

Response Format

The API returns a JSON object with the following structure: 

{
  "destination": "Lisbon",
  "price_range": "mid-range",
  "ideal_visit_times": [
    "April–May",
    "September–October"
  ],
  "top_attractions": [
    "Belém Tower",
    "Alfama District",
    "Jerónimos Monastery",
    "Tram 28"
  ]
}

Error Responses

  • 400 Bad Request: Missing or invalid destination parameter
  • 502 Bad Gateway: Error generating itinerary from the language model
